---

## 🚀 Deploy to Vercel via GitHub (Step-by-Step)

### STEP 1 — Open the project in VS Code

```bash
cd serenity-touch
code .
```

---

### STEP 2 — Initialize a Git repository

Open the VS Code terminal (`Ctrl + `` ` ``) and run:

```bash
git init
git add .
git commit -m "initial commit: Serenity Touch Wellness website"
```

---

### STEP 3 — Create a GitHub repository

1. Go to [github.com](https://github.com) and sign in
2. Click the **+** icon → **New repository**
3. Name it `serenity-touch` (or anything you like)
4. Set it to **Public** or **Private**
5. Do **NOT** check "Add a README file" (you already have one)
6. Click **Create repository**

---

### STEP 4 — Push your code to GitHub

GitHub will show you commands after creating the repo. Run these in your VS Code terminal:

```bash
git remote add origin https://github.com/YOUR_USERNAME/serenity-touch.git
git branch -M main
git push -u origin main
```

> Replace `YOUR_USERNAME` with your actual GitHub username.

---

### STEP 5 — Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in (use your GitHub account)
2. Click **Add New → Project**
3. Find and select your `serenity-touch` repository
4. Vercel auto-detects it as a static site — **no configuration needed**
5. Click **Deploy**

✅ Your site will be live at `https://serenity-touch.vercel.app` (or similar) in about 30 seconds.

---

### STEP 6 — Future updates (push = auto-deploy)

Every time you make changes in VS Code:

```bash
git add .
git commit -m "your message here"
git push
```

Vercel automatically re-deploys on every push to `main`. 🎉
